📝 About The Project
-----------------------------------------
Momentum은 크롬에서 새로운 탭을 열 때 나타나는 창입니다. 배경 화면과 한 줄 명언은 랜덤으로 바뀌게 구성하였고, 위치 액세스를 허용하면 해당 지역의 날씨를 간단하게 알려줍니다. 'What is your name?'란에 이름을 입력한 후 Log In을 누르면 인삿말을 받을 수 있으며, To-Do 리스트를 작성 및 삭제할 수 있습니다.
